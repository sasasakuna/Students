

This is a demo of spring mvc web project with gradle.  

It will contain a very simple example of Hello world as well as the tests.

Usage
Under the project root directory and input the 'gradle jettyRun' to start the server
Then input 'http://localhost:8080/SpringMVC/passedStudents' in browser to show the passed students.
Or inout 'http://localhost:8080/SpringMVC' in browser to show 'Hello'
